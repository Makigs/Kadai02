よくあるエラー

  • JQueryを記載したのに、なぜか動かない。エラーもでていない。

    +

    以下のようなケースが多いです。もう一度JQuery構文を見直しましょう
    -.（クラス）、#（ID）は記載した？
    -IDなのに.、クラスなのに#になっていない？

  • caught ReferenceError: XXX is not defined

    +

    定義されていない変数を呼び出しています。XXXで記載した変数・クラス・IDについて、以下を確認してください。
    名前のスペルミスがないか、”.””＃”は記載しているか、（JQueryセレクタの場合）カンマを入れているか

  • caught SyntaxError: Unexpected string

    +

    コードの構文に誤りがあることを示します。JQueryの記載について、以下を確認してください。
    括弧の閉じの欠落が内科、中括弧や他の文字の不適切な利用がないか。「よくある構文をみる」ページで文法を確認しましょう
