Linuxコマンド

分類 コマンド 操作の説明 コマンド記載例
"ディレクトリ"
移動・表示		 cd 指定したディレクトリ(フォルダ階層)へ移動 cd ubuntu
cd public_html		 →ubuntuフォルダへ移動
→public_htmlフォルダへ移動
cd ../ 1階層上のディレクトリ(フォルダ階層)へ移動 cd ../
cd ../../		 →1階層上へ移動
→2階層上へ移動
ls 今作業しているディレクトリ(フォルダ階層)のフォルダ/ファイル一覧を表示 ls
pwd 現在アクセスしているディレクトリ(フォルダ階層)がどこなのかを表示 pwd
"ファイル"
コピー・削除・
移動・表示		 cp 対象のファイルをコピー cp コピー元ファイル名 コピー先ファイル名
cp test01.html test02.html
→「test01」というファイルをコピーし「test02」で保存
rm 対象のファイルを削除 rm text01 →「test01」というファイルを削除
mv ファイルを移動 mv 移動元ファイル名 移動先フォルダ名/移動先ファイル名
mv test01.html public_html/test01.html
mv test01.html public_html/test09.html
→「test01」ファイルpublic_htmlへ移動
→「test01」ファイルpublic_htmlへ「text09」に名前を変更し移動
cat テキストファイルの内容を表示 cat test01.html →「test01」というファイルの内容を表示
"フォルダ"
作成・削除		 mkdir 新しいフォルダを作成 mkdir test →「test」というフォルダを新規
rmdir フォルダを削除 rmdir test →「test」というフォルダを削除
"テキスト編集"
起動・編集・保存		 vi テキストファイルの編集 vi test.html →「test」というファイルを編集
[i] （viのオプション）INSERTモード、書き込みが可能 vi test.html → i ファイルを編集モードへ
[Escキー] （viのオプション）INSERTモードを終了し、書き込み不可へ vi test.html → Escキー ファイルの編集モードを終了
[:] （viのオプション）コマンドモードへ切り替え vi test.html → ：キー コマンドモードへ切り替え
[:wq] （viのオプション）保存して終了へ vi test.html → :wq ファイルの編集を保存して終了
[:q] （viのオプション）終了 vi test.html → :q ファイルの編集を保存せず終了
[:q!] （viのオプション）強制的に終了 vi test.html → :q! viエディタを強制的に終了
